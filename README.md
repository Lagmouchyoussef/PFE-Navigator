# PFE-Navigator

PFE-Navigator is an institutional management platform for academic final-year projects (PFE). It bridges students, supervisors, and juries through specialized dashboards, real-time tracking, automated scheduling, and advanced analytics. A professional React & Bootstrap solution for seamless graduation workflows.

## Features

- **Multi-Role Dashboards**: Specialized interfaces for Students, Supervisors, Jury members, and Admins.
- **Project Tracking**: High-fidelity progression tracking with interactive timelines.
- **Document Management**: Centralized repository with version control for reports and deliverables.
- **Scheduling**: Automated planning for meetings and final defenses.
- **Analytics**: Real-time performance metrics and progress visualization.

## Tech Stack

- **Frontend**: React, Bootstrap 5, Recharts, Framer Motion, Lucide Icons.
- **Styling**: Vanilla CSS with a focus on high-fidelity SaaS aesthetics.
- **Architecture**: Modern component-based structure with centralized state management.

## Installation

1. Clone the repository
2. Navigate to the `Frontend` directory: `cd Frontend`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
